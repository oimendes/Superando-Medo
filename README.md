# 📚 Landing Page: Superando o Medo de Falar Inglês
Link: https://oimendes.github.io/Superando-Medo/
Este repositório contém o código-fonte de uma landing page de alta conversão desenvolvida para o projeto do **Franco**. O objetivo principal é oferecer uma interface limpa, rápida e persuasiva para profissionais que buscam destravar o inglês.

## 🚀 Funcionalidades
* **Design de Alta Conversão**: Estrutura baseada em gatilhos mentais e clareza visual.
* **Responsividade Total**: Experiência otimizada para Desktop, Tablets e Mobile.
* **Call to Action (CTA)**: Botões estratégicos conectados diretamente ao WhatsApp para suporte e vendas.
* **Navegação Slim**: Menu fixo e leve para facilitar a leitura.

## 🛠️ Tecnologias
* **HTML5** (Semântica e SEO)
* **CSS3** (Flexbox, Grid e Animações de Hover)
* **FontAwesome** (Ícones profissionais)

## 🎨 Referência Visual
O projeto foi construído seguindo uma identidade visual sóbria e moderna, focada em transmitir confiança e profissionalismo, utilizando uma paleta de cores equilibrada e tipografia marcante.
